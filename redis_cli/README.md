# redis_cli
A simple redis cli written by [go-netty](https://github.com/go-netty/go-netty)

### Preview
```bash
connecting redis server ...
connected
192.168.212.212:6379>get name
(empty)
192.168.212.212:6379>set name go-netty
OK
192.168.212.212:6379>get name
go-netty
192.168.212.212:6379>exit
exited
```
